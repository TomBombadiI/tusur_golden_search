# Метод золотого сечения

Учебный проект в рамках **учебной (ознакомительной) практики** по направлению  
**09.03.04 - Программная инженерия**.

Проект представляет собой реализацию **метода золотого сечения** для поиска экстремума функции на заданном интервале без использования производных.

---

## ?? О проекте

Метод золотого сечения - численный алгоритм оптимизации, предназначенный для поиска **минимума или максимума унимодальной функции** на отрезке `[a, b]`.

В рамках проекта реализовано:

- чистая функция алгоритма без ввода/вывода;
- консольный интерфейс (CLI);
- набор тестовых функций;
- модульные тесты;

---

## ?? Цель работы

Изучить и реализовать метод золотого сечения для поиска экстремума функции,  
а также закрепить навыки:

- реализации численных методов;
- проектирования модульной архитектуры;
- написания тестируемого кода;
- работы с CLI и автоматическими тестами.

---

## ?? Кратко о методе

Метод золотого сечения последовательно сужает интервал поиска экстремума,  
используя соотношение:

```
? = (1 + √5) / 2 ? 1.618
```

Алгоритм:
- не требует вычисления производных;
- работает только для **унимодальных функций**;
- применим как для поиска минимума, так и максимума.

## ?? Основная функция

```python
def golden_section_search(
    f: Callable[[float], float],
    a: float,
    b: float,
    eps: float = 1e-6,
    find_min: bool = True
) -> float:
```

**Параметры:**
- `f` - оптимизируемая функция;
- `a`, `b` - границы интервала;
- `eps` - точность;
- `find_min` - поиск минимума (`True`) или максимума (`False`).

## Установка

Для запуска CLI требуется зависимость `sympy`.

Windows:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Linux/macOS:

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Для тестов:

```bash
pip install -r requirements-dev.txt
```

---

## ?? Запуск CLI

Из корня проекта:

```bash
python -m src.cli
```

Можно выбрать готовую функцию из списка или ввести свою.
Поддерживаются: `sin`, `cos`, `tan`, `log`, `sqrt`, `exp`, `abs`, `pi`, `e`.
