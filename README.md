# Метод золотого сечения для решения уравнения

Учебный проект в рамках **учебной (ознакомительной) практики** по направлению  
**09.03.04 - Программная инженерия**.

Проект представляет собой реализацию **метода золотого сечения** для решения нелинейного уравнения `f(x) = 0` на заданном интервале.

---

## О проекте

Метод золотого сечения в данном проекте используется как способ уточнения корня уравнения на отрезке `[a, b]`.

В рамках проекта реализовано:

- чистая функция алгоритма без ввода/вывода;
- консольный интерфейс (CLI);
- набор тестовых функций;
- модульные тесты.

---

## Цель работы

Изучить и реализовать метод золотого сечения для решения уравнения `f(x)=0`,  
а также закрепить навыки:

- реализации численных методов;
- проектирования модульной архитектуры;
- написания тестируемого кода;
- работы с CLI и автоматическими тестами.

---

## Кратко о методе

Метод золотого сечения последовательно сужает интервал, содержащий корень,  
используя соотношение:

```
phi = (1 + sqrt(5)) / 2 ≈ 1.618
```

Алгоритм:
- не требует вычисления производных;
- требует, чтобы на концах отрезка значения были разных знаков;
- остановка по длине интервала.

## Основная функция

```python
def golden_section_root(
    f: Callable[[float], float],
    a: float,
    b: float,
    eps_x: float = 1e-6,
    max_iter: int = 1000
) -> tuple[float, int]:
```

**Параметры:**
- `f` - функция, для которой ищется корень;
- `a`, `b` - границы интервала;
- `eps_x` - точность по x;
- `max_iter` - ограничение по числу итераций.

Для запуска CLI требуется зависимость `sympy`.

```bash
pip install -r requirements.txt
```

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
